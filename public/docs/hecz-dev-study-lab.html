starter kit

Your study loop deserves its own universe.

Fork a real app, build your own question bank, import it, and turn notes into a feedback loop you can actually use.

Open lab hub Download class pack Fork the repo Download deck

Start with the app, not a blank page.

The public repo is a stripped starter. It gives you the engine, importer, local progress, review loop, and docs. You bring the bank.

Clone

Run the app locally and see the tiny starter bank.

Create

Write a small original bank from notes and allowed resources.

Import

Upload JSON or CSV in local development or an enabled fork, then practice and revise after misses.

First run.

git clone https://github.com/h3cz/study.git
cd study
npm install
npm run dev

Then open the local URL from Next.js, usually http://localhost:3000. Bank import is enabled automatically in local development.

Make the first bank tiny.

Minimum useful drop:

5 MCQs

Scenario-first questions with one clearly best answer.

3 flashcards

Vocabulary, ports, commands, or concepts you keep missing.

1 source note

Track where the idea came from and whether you can reuse it.
