# Study Bank Class Pack

Use this pack to draft a small original bank before importing it into hecz / study.

Files:

- `bank-template.json` - JSON format for questions, flashcards, PBQs, and acronyms.
- `questions-template.csv` - simple CSV for multiple-choice questions.
- `source-checklist.md` - source hygiene checklist.
- `peer-review-checklist.md` - review checklist for classmates.

Recommended flow:

1. Pick one narrow topic.
2. Draft 5-10 original questions.
3. Add source notes.
4. Peer-review with a classmate.
5. Import into `/import` in local development or an enabled lab fork.
6. Run a short quiz and revise confusing items.
