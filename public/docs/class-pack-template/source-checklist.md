# Source Checklist

Before importing a question, confirm:

- I wrote the question in my own words.
- I did not copy an exam dump, leaked question, or paid question bank.
- I can name the source or lab that taught the concept.
- The source is my notes, instructor-approved material, public docs, or openly licensed content.
- The source note includes title, author/organization, URL when available, and license.
- The explanation teaches the concept rather than only naming the answer.

When unsure, ask the instructor before importing or sharing.
